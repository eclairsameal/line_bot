# line_bot

## Development

```
pip install -r requirements.txt
```

## Environment Variable

.env
```
LINE_CHANNEL_ACCESS_TOKEN=
LINE_CHANNEL_SECRET=
OPENAPI_KEY=
```

## ngrok
https://ngrok.com/


* install
```
brew cask install ngrok
```

* run
```
ngrok http --authtoken [Authtoken] --host-header=rewrite 8080
```
